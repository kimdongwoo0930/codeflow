## 1. Product Vision

CodeFlow는 알고리즘 문제를 **"정답 맞히기"에서 끝내지 않는다.**

코드를 실행하고 → 그 과정이 어떻게 흘러가는지 눈으로 보고 → AI가 다음 학습 단계까지 연결해 주는 **인터랙티브 학습 플랫폼**을 지향한다.

| 핵심 가치             | 의미                                            |
| --------------------- | ----------------------------------------------- |
| **Process First**     | 결과보다 과정을 이해하게 하는 학습 경험         |
| **Personalized Flow** | 내 약점에 맞춰 다음 문제를 이어주는 학습 루프   |
| **Visual + AI**       | 코드 실행 과정과 개념 설명이 분리되지 않는 제품 |

> [!IMPORTANT]
> CodeFlow의 핵심은 **"많이 푸는 것"이 아니라 "왜 이렇게 동작하는지 이해하는 것"**이다.

---

## 2. Why Now

기존 알고리즘 학습은 두 방향에 치우쳐 있다.

- 문제를 **많이 푸는** 방식
- 정답 코드를 **외우는** 방식

하지만 초보자에게 더 중요한 것은 **"왜 이 코드가 여기서 이렇게 움직이는가"** 를 이해하는 일이다.

CodeFlow는 바로 그 지점을 겨냥한다. 아래를 모두 눈으로 보여주고, AI가 그 과정을 설명하면서 학습자의 사고를 다음 문제로 이어준다.

| 시각화 대상       | 설명                     |
| ----------------- | ------------------------ |
| `Current Line`    | 현재 실행 중인 코드 라인 |
| `Stack / Heap`    | 메모리 상태 변화         |
| `Loop / Branch`   | 반복·분기 흐름           |
| `Array / Pointer` | 인덱스와 포인터 이동     |
