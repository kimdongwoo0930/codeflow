# CodeFlow 기능 명세

---

## 회원 / 인증

- 이메일 회원가입
- 이메일 인증 (링크 클릭으로 계정 활성화)
- 이메일 로그인
- GitHub 소셜 로그인
- 로그아웃
- 토큰 자동 갱신
- 내 정보 조회

---

## 코드 실행 / 시각화

- 코드 제출 (Java / Python)
- 라인별 실행 상태 추출
- 실행 결과 실시간 스트리밍 (SSE) -> 실행이 끝난후 한번에 가져와 표현하기
- 현재 실행 라인 하이라이트
- Stack / Heap 상태 시각화
- Output 표시
- 실행 에러 위치 및 메시지 표시
- 실행 시간 / 메모리 제한 (샌드박스)

---

## 문제

- 문제 목록 조회 (주제 / 난이도 / 언어 필터)
- 문제 상세 조회
- 예제 테스트케이스 확인
- 코드 작성 및 제출
- 테스트케이스 통과 여부 확인

---

## AI

- 실행 과정 단계별 설명 생성
- 단계형 힌트 제공 (개념 → 접근 → 코드 방향 → 정답)
- 다음 문제 자동 추천
- AI 문제 생성 (주제 / 난이도 기반)
- 생성된 문제 자동 검증 (중복, 일관성, 예제 정확성)
- 검증 실패 시 기본 문제 fallback => 재 생성및 재검사 

---

## 학습 기록 (로그인 유저)

- 풀이 기록 저장
- 맞힌 문제 / 틀린 문제 구분
- 최근 푼 문제 목록
- 내가 저장한 문제 목록
