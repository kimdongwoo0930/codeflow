문제 설명

N×M 크기의 미로가 주어집니다. 시작 위치 (1, 1)에서 출발하여 출구 (N, M)까지 이동하는 최단 경로의 길이를 구하세요.

미로에서 1은 이동할 수 있는 칸, 0은 이동할 수 없는 벽을 나타냅니다. 이동은 상하좌우 4방향만 가능합니다.

입력 형식
Input
첫째 줄에 N, M (2 ≤ N, M ≤ 100)
이후 N줄에 걸쳐 M개의 정수 (0 또는 1)
출력 형식
Output
최단 경로의 칸 수를 출력
(시작, 도착 칸 포함)

제약 조건
  • 2 ≤ N, M ≤ 100
  • 시작점 (1,1)과 도착점 (N,M)은 항상 1
  • 항상 도달 가능한 입력만 주어짐
💡 힌트

BFS는 가중치 없는 그래프에서 최단 경로를 보장합니다. Queue에 현재 위치와 이동 횟수를 함께 저장하고, 방문 배열로 중복 방문을 막으세요.

21 static int bfs(int[][] maze, int n, int m) {
22 boolean[][] visited = new boolean[n][m];
23 Queue<int[]> q = new LinkedList<>();
24 q.add(new int[]{0, 0, 1});
25 visited[0][0] = true;
26
27 while (!q.isEmpty()) {
28 int[] cur = q.poll();
29 int x = cur[0], y = cur[1], dist = cur[2];
30
31 if (x == n - 1 && y == m - 1) return dist;
32
33 for (int d = 0; d < 4; d++) {
34 int nx = x + dx[d], ny = y + dy[d];
35 if (nx < 0 || ny < 0 || nx >= n || ny >= m) continue;
36 if (maze[nx][ny] == 0 || visited[nx][ny]) continue;
37 visited[nx][ny] = true;
38 q.add(new int[]{nx, ny, dist + 1});
39 }
40 }
41 return -1;
42 }
